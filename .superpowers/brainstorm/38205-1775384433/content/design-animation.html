Animation Cycle

Slot-machine easing over 5 seconds
0.0s Alice → Bob → Carol → Dave → ~50ms per swap (fast)
2.0s Eve → Frank → ~200ms per swap
3.5s Alice → ... → ~500ms per swap (slowing)
4.5s Dave → ~1000ms (almost stopped)
5.0s ★ BOB ★ Winner! Flash/pulse effect
Winner is pre-determined at Start. The easing function (cubic ease-out) controls the deceleration. Names are picked randomly during cycling but the sequence is rigged to land on the winner.

Character Palette

Built once on load, cached for animation
Charset: .,:;!+-=*#@%&abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789
Fonts: Georgia × [300, 500, 800] × [normal, italic] = 6 variants
Per character: measure width (pretext) + estimate brightness (canvas) → sorted by brightness
Lookup: binary search for target brightness, then score nearby candidates by brightness + width match
